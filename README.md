## 👋 About me

Hi, I'm **Gaston** — an **SDET &amp; DevOps Engineer** from Argentina. I design robust
test‑automation frameworks and CI/CD pipelines, and I care about quality end to
end: from cloud infrastructure down to the last edge case. I've had the privilege
of working with teams at **Globant, Caylent, CLARO, Naranja X, Coca‑Cola Andina**
and more.

Away from the terminal I'm a **photographer and an athlete**. I shoot light, motion
and people, and I track my own training and health as a long‑running data project.
This portfolio is where both sides of me live in the same place.

## 🌐 gastongenaud.com — one site, three modes

A single switch (`dev · studio · greek`) flips the entire site between three worlds:

| Mode | What you'll find |
| :--- | :--- |
| 🖥️ **Dev** | The engineer portfolio — stack, experience and projects. |
| 📷 **Studio** | An editorial photo gallery — 95 frames straight from Lightroom. |
| 👑 **Greek** | _Athletic Transformation_ — a data‑driven body‑recomposition log. |

## 🛠️ Run it locally

Built with **Next.js 13 (App Router) + TypeScript + Tailwind**, deployed on **Vercel**.

```bash
git clone https://github.com/GastonAGenaud/portfolio.git
cd portfolio
npm install
npm run dev   # → http://localhost:3000
```
